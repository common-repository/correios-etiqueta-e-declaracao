Gera etiqueta para envio pelos correios e declaração de conteúdo.
== Description ==
Gere a etiqueta para envio pelos correios e a declaração de conteúdo com apenas um clique.
Ganhe mais produtividade para o envio de suas encomendas.
Agora também com integração com o correios para envio de PLP para assiantes.
== Installation ==
Instale e insira os dados do remetente.
== Screenshots ==
1. Tela para configuração dos dados do remetente.
2. Tela de pedidos, com o botão para emitir a etiqueta (precisa estar no status processando e a ações da tela visíveis).
3. Tela de impressão da etiqueta e declaração de conteudo.
4. Botão de emitir etiqueta na tela dos detalhes do pedido.
